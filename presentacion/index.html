Tema


Este Proyecto esta diseñado por el momento de forma visual, para que el propietario destinado pueda tener una idea de como se verá nuestro proyecto de acuerdo a las necesidades que el propietario nos ha proporcionado, en este caso, los temas a enfocar serán un sistema punto de venta y uno de sistema de control de citas para el usuario. Para el sistema punto de venta se tiene pensado que el propietario pueda tener un control de los productos que se venden en su negocio, que en este caso es un Taller de Servicio Tecnico y a la vez venta-compra de electrodomesticos (refrigeradores, lavadoras, licuadoras, etc). para el sistema de control de citas se tiene pensado que el usuario se de de alta en la plataforma y pueda agendar una cita para el servicio que su producto requiera, de esta forma el propietario podrá tener un control de los servicios que se requieren y los que se han realizado.
